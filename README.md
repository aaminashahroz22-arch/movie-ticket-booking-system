Movie Ticket Booking System (Java)

A simple **console-based Movie Ticket Booking System** developed in **Java** using Object-Oriented Programming concepts.  
This project simulates a basic movie ticket platform where users can view available seats, book one or more seats, and see booked seats marked as **X**.



Features ──

Display seats in theatre format  
Available seats shown by seat numbers  
Booked seats shown as **X**  
Book multiple seats at once  
Prevent duplicate booking  
Invalid seat number handling  
Menu-driven program
